# Docs about .NET SingalR


## Vue client

### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```

#### Install SignalR client for Vue
```bash
npm i @microsoft/signalr@latest --save
```
